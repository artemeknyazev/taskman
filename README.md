# Taskman

Taskman (from *Task* *Man*ager) is a sample task (currently just todo) management app built using React, Redux and Express

## Installation

```bash
# Clone it
> git clone https://github.com/artemeknyazev/taskman.git
> cd taskman

# Generate sample database
> bash db/run.sh

# Build
> npm run build

# Starts
> npm start
```
